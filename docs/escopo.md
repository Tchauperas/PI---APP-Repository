# Escopo do Sistema – FinansMe

## Descrição Geral

O FinansMe é um aplicativo móvel voltado para o controle e organização de finanças pessoais. 
O sistema permite que usuários registrem suas transações financeiras, acompanhem seus gastos 
e visualizem sua situação financeira de forma centralizada.

O aplicativo será desenvolvido utilizando o framework Flutter, permitindo execução em 
dispositivos Android e iOS. O backend será desenvolvido em Node.js com TypeScript, 
expondo uma API REST responsável pelo gerenciamento dos dados dos usuários.

---

## Objetivo do Sistema

O objetivo do FinansMe é ajudar usuários a:

- Registrar receitas e despesas
- Acompanhar seu histórico financeiro
- Visualizar sua situação financeira de forma clara
- Organizar melhor sua vida financeira

---

## Funcionalidades Incluídas

O sistema incluirá as seguintes funcionalidades:

- Cadastro de usuários
- Login e autenticação
- Registro de transações financeiras
- Visualização de histórico de transações
- Visualização de resumo financeiro
- Gerenciamento de contas ou bancos
- Perfil do usuário
- Navegação entre telas do aplicativo

Principais telas do sistema:

- Login
- Home
- Transações
- Lançamento de transação
- Bancos
- Perfil

---

## Fora do Escopo

O sistema não incluirá:

- Integração automática com bancos
- Pagamentos ou transferências financeiras
- Operações financeiras reais
- Versão web da aplicação
