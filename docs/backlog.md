# Product Backlog – FinansMe

O backlog do produto organiza funcionalidades em **épicos** e **user stories**.

---

# Épico 1 – Autenticação

## US01 – Cadastro

Como usuário  
Quero criar uma conta no aplicativo  
Para começar a registrar minhas finanças.

## US02 – Login

Como usuário  
Quero realizar login no sistema  
Para acessar minhas informações financeiras com segurança.

---

# Épico 2 – Gerenciamento de Transações

## US03 – Registrar Transação

Como usuário  
Quero registrar uma nova transação financeira  
Para controlar meus gastos e receitas.

## US04 – Classificar Transação

Como usuário  
Quero classificar minhas transações como receita ou despesa  
Para organizar melhor minhas finanças.

## US05 – Editar ou Remover Transação

Como usuário  
Quero editar ou excluir uma transação  
Para corrigir dados registrados incorretamente.

---

# Épico 3 – Visualização Financeira

## US06 – Histórico de Transações

Como usuário  
Quero visualizar meu histórico de transações  
Para acompanhar meus gastos ao longo do tempo.

## US07 – Resumo Financeiro

Como usuário  
Quero visualizar um resumo da minha situação financeira  
Para entender rapidamente minhas receitas e despesas.

---

# Épico 4 – Gerenciamento de Contas

## US08 – Contas ou Bancos

Como usuário  
Quero visualizar minhas contas ou bancos cadastrados  
Para organizar minhas fontes financeiras.

---

# Épico 5 – Perfil do Usuário

## US09 – Gerenciar Perfil

Como usuário  
Quero visualizar e editar meu perfil  
Para manter minhas informações atualizadas.
